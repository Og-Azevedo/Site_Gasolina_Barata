Sobre
Projeto desenvolvido por Og Azevedo

O site Gasolinabarata.online tem como ojetivo principal ajudar a população de Maceió-Alagoas a encontrar o posto de gasolina mais barato da capital. Na sua tela principal o usuário encontra o ranking dos 10 melhores preços em tempo real.

O desenvolvimento do back-end foi feito na linguagem Python, utilizando o framework Flask. Toda o front-end foi desenvolvido usando a ferramenta open source Bootstrap. Para fornecer essa informação o site conta com as informações fornecidas pelas API da Sefaz - AL.
